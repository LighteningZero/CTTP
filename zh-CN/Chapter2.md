# CTTP 教室纸条文本传输协议 - Chapter 2

标签： CTTP LighteningZero CTTP-v1.0

---

1. 通用规定
    1. 为了使纸条通信高效（我们深知传统纸条通信的传输延迟之大），必须设计一个无需“握手”的协议。
    2. 为标识协议和其所使用的版本，报头开头必须为：
        ```text
        CTTP / 1.0
        ```
    3. 我们深知 IP及TCP 协议之不便，所以 CTTP 将不基于 TCP/IP 。
    4. 我们规定报文中要有“标记位”，用于标识消息类型，如：
        ```text
        CTTP / 1.0  P2PSND
                    ^~~~~~
                    标记位
        ```
